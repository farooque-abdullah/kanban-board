# 📋 TaskFlow — Kanban Board

A fully functional **drag-and-drop Kanban board** built with React. Manage tasks across sprint columns with real-time updates, priorities, and tags.

## ✨ Features

- 🖱️ **Drag & drop** tasks between columns (native HTML5 DnD)
- ➕ **Add tasks** with custom tags and priority levels
- 🗑️ **Delete tasks** with one click
- 🌙 **Dark / Light mode** toggle
- 📊 **Progress tracker** — % completion at a glance
- 🎨 **Color-coded tags** — Frontend, Backend, Design, Testing, DevOps, etc.
- 🔴🟡🟢 **Priority indicators** — high, medium, low

## 🛠️ Tech Stack

- **React 18** with hooks (useState)
- **HTML5 Drag and Drop API** — zero dependencies for DnD
- **Vite** — dev server & bundler
- Zero external UI libraries

## 🚀 Getting Started

```bash
git clone https://github.com/yourusername/kanban-board.git
cd kanban-board
npm install
npm run dev
```
